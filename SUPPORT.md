# Suporte

Precisa de ajuda com o Caderno? Esta página reúne os canais certos para cada tipo
de situação. Escolher o canal certo ajuda a gente a responder mais rápido. 🙂

> O Caderno é mantido em **trabalho voluntário**. Fazemos o possível para
> responder, mas não há prazos garantidos — agradecemos a paciência.

## Por onde começar

Antes de abrir qualquer coisa, dá uma olhada em:

- **[README][readme]** — visão geral, o que o Caderno faz e como rodar.
- **[Documentação][docs]** — arquitetura e guia de desenvolvimento.
- **Issues já existentes** — talvez sua dúvida ou bug já tenha sido relatado.

## Qual canal usar

| Situação | Onde ir |
| --- | --- |
| 🐛 Encontrou um bug | Abra um [issue de bug][issues] usando o template |
| 💡 Tem uma ideia / pedido de funcionalidade | Abra um [issue de funcionalidade][issues] |
| ❓ Dúvida de uso ou quer discutir uma ideia | [GitHub Discussions][discussions] |
| 🔒 Vulnerabilidade de segurança | **Não abra issue pública** — siga o [SECURITY.md][security] |
| 🤝 Quer contribuir com código | Leia o [CONTRIBUTING.md][contributing] |
| 💛 Quer apoiar o projeto | [GitHub Sponsors][sponsor] |

## Reportando bugs

Bons relatos de bug aceleram a correção. Inclua:

- O que você esperava que acontecesse e o que aconteceu.
- Passos para reproduzir.
- Ambiente (navegador, sistema, versão/commit do Caderno).

O template de issue de bug já pede essas informações.

## Apoiando o projeto

O Caderno é livre, gratuito e sem lock-in — e a intenção é que continue assim.
Apoiar é **opcional e sempre será**: quem contribui sustenta o projeto para todo
mundo, não compra vantagem. Se ele te ajuda, considere
[ser um apoiador][sponsor]. 💛
